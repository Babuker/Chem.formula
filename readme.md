# Chemical Formula Optimizer

## Overview
A web-based tool for designing optimal chemical formulations with economic considerations. The tool supports both English and Arabic interfaces.

## Features
- Dual language support (English/Arabic)
- Automatic and manual excipient selection
- BP/USP pharmacopeia compliance
- Multiple product forms (tablets, capsules, syrups, powders)
- Cost optimization algorithms
- Quality specifications display
- Export and save functionality
- AI-powered suggestions
- Interactive pie charts
- PDF export capability

## Live Demo
Visit: https://babuker.github.io/Chem.formula/

## Installation
1. Download all files to a folder
2. Open `index.html` for English version
3. Open `index-ar.html` for Arabic version
4. No server requirements - works directly in browser

## Usage
1. Enter basic information (formula name, reference, product form)
2. Add active ingredients
3. Choose excipient input method (manual or automatic)
4. Set design goals and budget
5. Click "Design Optimal Formula"
6. Review results and export as needed

## File Structure
- `index.html` - English interface
- `index-ar.html` - Arabic interface
- `style.css` - Common styles
- `script.js` - Core JavaScript logic
- `manifest.json` - PWA manifest
- `.nojekyll` - GitHub Pages configuration
